# Python-GUI-DISKicon
Python GUI 程序，快速制作个性化磁盘图标

### 9:41PM, Mar 18th, 2018 @ HDU_Wireless

## 使用说明

* 更改磁盘或U盘的默认图标
* 本质上是往磁盘中写入一个 autorun.inf 文件
